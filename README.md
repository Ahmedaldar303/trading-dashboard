# 📊 Should I Be Trading? — Bloomberg Terminal Market Dashboard

> Professional SPX 0DTE Trading Dashboard with Real-Time Market Scoring
> 
> Built for Saudi Arabia Time (AST) traders | Full Arabic/English bilingual support

## ✨ Features

- ✅ Real-time market scoring (0-100%)
- ✅ 5-component weighted scoring system
- ✅ Execution Window Score (setup quality evaluation)
- ✅ YES / CAUTION / NO decision engine
- ✅ Claude AI market analysis
- ✅ Bilingual Arabic/English interface (RTL/LTR)
- ✅ Bloomberg Terminal aesthetics
- ✅ Live data: SPY, QQQ, VIX, DXY, 10Y Treasury, all 11 sectors
- ✅ Auto-refresh every 45 seconds
- ✅ Responsive design (desktop/tablet/mobile)

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- Anthropic API key (free at https://console.anthropic.com/)

### Installation

```bash
# 1. Clone repository
git clone https://github.com/yourusername/trading-dashboard.git
cd trading-dashboard

# 2. Install dependencies
npm install

# 3. Configure environment
cp .env.example .env

# 4. Add your Anthropic API key to .env
# ANTHROPIC_API_KEY=sk-ant-your-key-here

# 5. Start the dashboard
npm start
